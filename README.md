# Code Sample
Simple iOS client app cosuming RESTful service,

### Features:
  * Movies list
  * Movie Detail
  * Sorted by popularity
  * Search
  * Cache using NSCache
  * Pagination
 
## Architecture

MVVM was used as the core architecture for this code.
